## 5) Withdrawals and conversions
### 5.1 Withdraw availability
The Withdraw button is shown only if:
1) GET /api/v1/admin/system/status returns withdrawalsActive: true
2) The user's total balance is at least 500 USDT equivalent (50 CROWN)

If not, the Withdraw button is hidden completely (display: none).

### 5.2 Minimum withdrawal
Minimum withdrawal is 50 CROWN.
If the user enters less than 50 CROWN, show a red validation error:
"Minimum withdrawal is 50 CROWN"

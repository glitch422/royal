# 06 - המרת CROWN ל-USDT (Withdraw)

תאריך תחולה: 2026-02-21

## זמינות
כפתור Withdraw מופיע רק אם withdrawalsActive=true וגם השווי הכולל ≥ 500 USDT (50 CROWN). אחרת מוסתר.

## מינימום
מינימום משיכה: 50 CROWN  
פחות מזה: "Minimum withdrawal is 50 CROWN"

## רשתות
TRC20, ERC20.

## תהליך
Wallet → Withdraw → סכום → רשת → כתובת → שליחה → סטטוסים pending/approved/rejected/sent עם TXID.
